vim-config
==========

Vim config and plugin files.


Setup:
```
1. Enter vim-config
2. ./setup.sh
3. Start vim and input ":PlugInstall" to install plugins.
```

Use vim-plug to manage the plugin

Plugin lists:
```
> taglist
> supertab
> SearchComplete
> vim-airline
> vim-gitgutter
> jedi-vim
```
