vim-config
==========

Vim config and plugin files.

THIS ONLY WORKS ON LINUX!!!

Setup:
```
1. Enter vim\_config
2. ./setup.sh
3. cd ~
```

Use submodule to manage the plugin

Plugin lists:
> pathogen
> SearchComplete
> supertab
> taglist
> mustache
> vim-gradle
> vim-airline

Dependence:
    ctags
    cscope
